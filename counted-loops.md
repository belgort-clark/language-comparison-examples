# Counted Loops

## Counted Loop in JavaScript

Examples of counted Loops that iterate 5 times

```javascript
for(let i=0; i<5; $i++){
    console.log(i);
}
```

## Counted Loop in Python

```python
for i in range(5):
    print(i)
```

## Counted Loop in PHP

```php
for($i=0;$i<5;$i++){
    echo $i;
}
```
