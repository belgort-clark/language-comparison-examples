# Functions with Arguments

## JavaScript

```javascript
function sayHello(person){
    console.log(`Hello ${person}`)
}

sayHello("Bruce")
```

## Python

```python
def sayHello(person):
    print(f"Hello {person}")

sayHello("Bruce")
```

## PHP

```php
function sayHello($person){
    console.log("Hello! $person");
}

sayHello("Bruce");
```
